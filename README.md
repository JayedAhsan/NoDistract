# NoDistract -  Block Facebook Reels Easily

NoDistract is a lightweight, privacy-focused Android application designed to block the Facebook Reels tab.

## How to use:

* Download: Get the latest APK from the [Releases](https://github.com/JayedAhsan/NoDistract/releases/) section.
* Install: Open the app and click the Enable button.
* Grant Permission: You will be prompted to grant Accessibility Permission to NoDistract.
* Focus: Toggle the service on/off at your convenience.
